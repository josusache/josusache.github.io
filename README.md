# josusache.github.io

Portfolio personal para Josu Gonzalez, fotografo, videografo y filmmaker, con foco en trabajos visuales que integran IA, foto y film. El dominio final previsto es `josugonzalez.com`.

## Vista local

Abre `index.html` directamente o sirve la carpeta con:

```bash
python -m http.server 8087
```

Despues visita `http://127.0.0.1:8087/`.
